## 原始來源
[ciffel/Ecourse-Fix](https://github.com/ciffel/Ecourse-Fix)

## 說明
修正中正大學教學課程平台(ecourse)網站的相容性問題及顯示效果。

## 修改項目
- 移除 ecourse 首頁上破壞排版的元素
- 在無標題的框架主頁增加標題以供辨識
- 修正表格四周裝飾性圖案的正確長度
- 修正功能選單連結無法在正確框架中開啟
- 修正課程教材的分項連結無法在正確框架中開啓
- 修正教師資訊中的下拉式選單無作用的問題
